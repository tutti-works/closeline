# CLOSELINE 概要

固定長の直線を交互に引き、交点から作られる自分のドットで三角形の陣地を獲得するゲームです。

## プレイヤー向けルール

- 初期盤面には中立線が3本あります。
- 人間とCPUが交互に同じ長さの線を引きます。
- 新しい線は既存線と1回以上交差する必要があります。
- 交点は後から線を引いたプレイヤーのドットになります。
- 自分のドット3点が、盤面上の実在する線で囲まれていると三角形を獲得します。
- 三角形の辺には中立線、自分の線、相手の線を利用できます。
- 獲得済み陣地の内部、辺、頂点には新しい線を引けません。
- 縦方向またはほぼ縦方向の線は禁止です。

## 操作方法

PCでは始点をクリックし、次に方向点をクリックして仮配置します。スマートフォンでは始点からドラッグして角度を決めます。線の長さは固定で、確定ボタンで配置します。キャンセルで仮配置を取り消せます。

## 勝敗条件

規定ターン終了、連続パス、双方に合法手がない場合に終了します。獲得した三角形の合計面積が大きいプレイヤーの勝利です。同点の場合は三角形数、最大三角形面積の順に判定します。

## 画面構成

盤面、スコア、現在ターン、CPU思考中表示、設定、ルール、リスタート、終了画面を表示します。
