# CLOSELINE 概要

CLOSELINEは、同じ長さの直線を交互に置き、線と盤面外枠で閉じた領域を獲得する陣取りゲームです。

## ルール

- すべての線は設定された同じ長さです。
- 新しい線は既存線と交差・重複できません。
- 線の端点だけは、既存線の端点、既存線の途中、盤面外枠へ接続できます。
- 線の途中が既存線や盤面外枠に触れる配置は無効です。
- 獲得済み領域の内側へ新しい線は置けません。
- 1手で複数の新規領域が閉じた場合は、すべて手番プレイヤーが獲得します。

## 勝敗条件

- 2人対戦では最大領域モードです。各プレイヤーが獲得した領域のうち、最も面積が大きい1領域を比較します。
- 3人以上では合計面積モードです。獲得済み領域の合計面積を比較します。
- 同点時は獲得領域数、それでも同じなら引き分けです。

## 操作

盤面を押して始点を決め、ドラッグ方向で線の角度を決めます。線の長さは常に設定値に固定されます。プレビューが青系なら配置可能、赤なら無効です。スマートフォンでも誤操作を避けるため、ドラッグ後に「配置」ボタンで確定します。

## 画面構成

中央に正方形の盤面、横または上部に現在手番、スコア、ターン数、リスタート、ルールボタンを表示します。スマートフォンでは縦画面を優先し、盤面操作中のページスクロールを抑制します。
